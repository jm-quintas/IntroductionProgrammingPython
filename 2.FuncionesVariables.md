# Funciones y Variables.

## Funciones predefinidas en Python.
Es una función que está disponible para su uso inmediatamente después de iniciar el intérprete de Python, sin necesidad de importar ningún módulo.

## Características Claves.
Estas funciones son esenciales y vienen con el lenguaje, proporcionando funcionalidades comunes y de propósito general.

- **Disponibilidad Universal**: siempre están accesibles en cualquier parte del código Python.

- **Eficiencia**: suelen estar implementadas en C y optimizadas, haciéndolas muy eficientes para tareas comunes.

- **Fundamentales**: proporcionan la base para muchas operaciones de programación esenciales.

  Ejemplos de funciones predefinidas:

  | Función | Descripción | Ejemplos |
  | :---: | :---: | :---: |
  | print() | Muestra datos en la consola (salida estándar) | print("Hola") - *Devuelve: Hola* |
  | type() | Devuelve el tipo de un objeto  | type(42) - *Devuelve: <class 'int'>* |

## Diferencia con las Funciones de Módulos.
Es importante distinguirlas de las funciones que requieren importación:

- **Predefinidas (Built-in)**: como *print()* o *len()*. No necesitan **import**.

- **De Módulos**: requieren una declaración *import* para ser utilizadas (ej. **math.sqrt()** requiere **import math**).

# Función print().
Es, quizás, la función predefinida más fundamental en Python. Su propósito principal es *mostrar una cadena de texto u objetos en la consola* o enviarlos a otro flujo de salida estándar. La sintaxis básica de la función **print()** es la siguiente:

```Python
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

- ***objects (Argumentos Posicionales)**: puedes pasarle cero o más objetos separados por comas. La función **print()** los convertirá a su representación en cadena antes de mostrarlos.

```Python
nombre = "NiClau"
edad = 11
print("Mi nombre es", nombre, "y tengo", edad, "años.")
# Salida: Mi nombre es NiClau y tengo 11 años.
```

## Parámetros Clave Opcionales:
Los argumentos posicionales son los datos a mostrar. Los argumentos de palabra clave (sep, end, file, flush) controlan cómo se muestra la salida.

- **sep(separador)**: define la cadena que se inserta entre los objetos mostrados. El valor por defecto es un espacio simple (' ').

```Python
print("Manzana", "Limon", "Pera", sep=('|'))
# Salida: Manzana | Limon | Pera
```
```Python
print(13, 11, 2025, sep=('-'))
# Salida: 13-11-2025
```

- **end (Finalizador)**: define la cadena que se añade al final de la salida después de que todos los objetos han sido mostrados. El valor por defecto es un salto de línea ('\n'), lo que hace que la siguiente llamada a **print()** comience en una nueva línea.

**Ejemplo 1:** comportamiento por defecto.
```Python
print("Línea 1")
print("Línea 2")
# Salida:
# Línea 1
# Línea 2
```

**Ejemplo 2:** cambiando el finalizador.
```Python
print("Línea 1", end=('/'))
print("Línea 2")
# Salida:
# Línea 1 / Línea 2
```

- **file (flujo de salida)**: se utiliza principalmente dentro de la función predefinida **print()** para especificar dónde debe enviarse la salida.

**Ejemplo 1:** salida Predeterminada (Consola).

```Python
import sys
# Estas dos líneas son equivalentes:
print("Hola mundo")
print("Hola mundo", file=sys.stdout)
# Salida:
# Hola mundo
# Hola mundo
```

**Ejemplo 2:** salida redefinida.

```Python
# Abre un archivo llamado 'log.txt' en modo de escritura ('w')
with open('log.txt', 'w') as archivo_log:
    # La salida se envía al objeto 'archivo_log' en lugar de a la consola
    print("Inicio del proceso...", file=archivo_log)
    print("Datos procesados exitosamente.", file=archivo_log)

# Después de que el bloque 'with' termina, el archivo se cierra automáticamente.
# El archivo 'log.txt' contendrá:
# Inicio del proceso...
# Datos procesados exitosamente.
```

# Función Input().
Es una función integrada que **se utiliza para obtener la entrada de datos del usuario** a través de la consola o la terminal.

## Uso Básico de input().
La función *input()* hace dos cosas principales:

- Muestra un mensaje (opcional) en la pantalla.

- Pausa la ejecución del programa y espera a que el usuario escriba algo y presione Enter.

- Devuelve la entrada del usuario como una cadena de texto (str).

```Python
input("Cual es tu nombre: ")
print("Hola NicLau")
# Salida:
# Cual es tu nombre: NicLau
# Hola NicLau
```

Es fundamental recordar que la función **input()** *siempre devuelve el valor como una cadena de texto (str)*, incluso si el usuario introduce números.

# Variables.
Son esencialmente *etiquetas o nombres simbólicos que se utilizan para almacenar y referenciar valores* en la memoria de la computadora. A diferencia de otros lenguajes de programación, *Python no requiere que declares explícitamente el tipo de dato de una variable*; el tipo se infiere dinámicamente cuando se le asigna un valor.

## Reglas para Nombrar Variables.
Hay tres reglas principales que debes seguir al nombrar variables en Python:

- **Caracteres Válidos**: solo pueden contener letras (a-z, A-Z), números (0-9) y el guion bajo (_).
- **No empezar con Números**: el nombre de una variable no puede comenzar con un número.

  - *Correcto*: mi_variable1

  - *Incorrecto*: 1variable
- **No usar palabras claves**: no puedes usar las palabras reservadas de Python (como **for**, **if**, **while**, **def**, etc.) como nombres de variables.

**Convención**: aunque Python permite mayúsculas, la convención de estilo recomendada (PEP 8) es usar minúsculas y separar las palabras con guiones bajos.
