# Historia de Python.

El lenguaje de programación **Python** fue creado por **Guido van Rossum**, un programador neerlandés, a finales de la década de 1980 en el Centrum Wiskunde & Informatica (CWI) en los Países Bajos. Se concibió como un proyecto personal para mantenerse ocupado durante las vacaciones de Navidad. El nombre *"Python"* proviene de la admiración de Van Rossum por el grupo de comedia británico Monty Python's Flying Circus.

## Cronología de Hitos Clave.

**1989 (Finales)**: Guido van Rossum comienza la implementación del lenguaje como sucesor del lenguaje ABC, buscando incluir el manejo de excepciones y la interacción con el sistema operativo Amoeba.

**1991 (Febrero)**: Se pública la primera versión del código, Python 0.9.0, en el grupo de noticias alt.sources. Ya incluía clases con herencia, manejo de excepciones y tipos de datos modulares como list, dict, y str.

**1994 (Enero)**: Se lanza Python 1.0. Introdujo características de programación funcional como lambda, reduce, filter y map. En este mismo año se formó comp.lang.python, el foro de discusión principal, marcando el crecimiento de la comunidad.

**2000 (Octubre)**: Llega Python 2.0. Marcó una nueva era al añadir mejoras significativas como la comprensión de listas (list comprehensions) y un sistema de recolección de basura para referencias cíclicas.

**2008 (Diciembre)**: Se lanza Python 3.0 (también llamado Python 3000 o Py3K). Esta versión fue una ruptura mayor con las anteriores (Python 2.x), ya que no es completamente compatible con el código antiguo. Su objetivo era corregir fallos de diseño del lenguaje, facilitando su mantenimiento y evolución a largo plazo. Entre los cambios notables estuvo la unificación de los tipos de cadena a Unicode y la transformación de print en una función.

**2010**: Se lanza Python 2.7, la última versión de la serie 2.x, con soporte extendido.

**2020 (Enero)**: Final oficial del soporte para Python 2.7, consolidando a la serie Python 3.x como el estándar y única rama con soporte activo.

## Filosofía y Popularidad.

Python ha ganado una inmensa **popularidad gracias a su sintaxis clara y legible**, que enfatiza la sencillez y la productividad (lo que se conoce como código "pythónico", influenciado por "El Zen de Python"). Actualmente, es uno de los lenguajes de programación más populares y versátiles del mundo, siendo fundamental en áreas como:

- **Ciencia de Datos y Aprendizaje Automático (ML)**: es el lenguaje dominante para estas áreas, con *herramientas clave como Pandas, Scikit-learn y TensorFlow*.

- **Desarrollo Web**: utilizado para la creación de backends (servidores) con *frameworks populares como Django y Flask*.

- **Automatización de Tareas (Scripting)**: se usa para escribir scripts que automatizan tareas repetitivas de administración de sistemas.

- **Análisis de Big Data**: utilizado en *entornos distribuidos como Apache Spark*.

# ¿Qué es Python?

Es un lenguaje de programación de alto nivel, interpretado, de propósito general, y orientado a objetos (aunque también soporta otros paradigmas).

## Características Principales.

- **Alto Nivel**: esto significa que el *código se asemeja más al lenguaje humano (inglés)* que al código de máquina, haciendo que sea más fácil de leer y escribir. No tienes que preocuparte por gestionar la memoria de bajo nivel.

- **Interpretado**: el código Python se *ejecuta línea por línea por un intérprete*, en lugar de ser compilado a código de máquina de antemano. Esto facilita la depuración y el prototipado rápido.

- **De Propósito General (Versatilidad)**: se utiliza en una vasta variedad de dominios, incluyendo desarrollo web, análisis de datos, inteligencia artificial, automatización de sistemas, y desarrollo de software de escritorio.

- **Fácil de Leer (Sintaxis Clara)**: su filosofía de diseño enfatiza la legibilidad y la sencillez de la sintaxis, utilizando la sangría obligatoria para definir bloques de código (en lugar de llaves o corchetes), lo que resulta en un código más claro y menos propenso a errores.

- **Orientado a Objetos (OO)**: permite *organizar el código utilizando clases y objetos*, facilitando la modularidad y la reutilización del código.

- **Gran Biblioteca Estándar y Ecosistema**: Python viene con una *enorme biblioteca estándar preconstruida y cuenta con un vasto ecosistema de módulos y frameworks de terceros* (como NumPy, Pandas, Django, y TensorFlow) que amplían enormemente su funcionalidad.
